The requirements

 
var fs = require('fs'),
  gulp = require('gulp'),
  gutil = require('gulp-util'),
  jade = require('gulp-jade');

Just your basic node require modules.

  • fs is for node files system commands.
  • gulpjs is the task managed
  • gulp-util adds a little more functionality to gulp. I mostly use it for error logging
  • Jade Templates are my preferred markup template system
